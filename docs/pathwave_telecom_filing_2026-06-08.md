# PathWave — 통신판매업 신고 가이드

> **작성일**: 2026-06-08
> **대상 서비스**: PathWave (운영사: 주식회사 트리거소프트 / TRIGGER SOFT CO., LTD.)
> **신청 경로**: [정부24 통신판매업 신고](https://www.gov.kr/portal/onestopSvc/comm)
> **선결 조건**: 사업자등록증 발급 완료 (사용자 보유 확인)

---

## 0. 신청 흐름 한눈에

```
[정부24 로그인 (공동인증서)]
   ↓
[통신판매업 신고서 작성 — 본 문서의 입력값 그대로]
   ↓
[첨부 서류 업로드: 사업자등록증 + 법인등기부 + 구매안전서비스 이용확인증(또는 면제 사유)]
   ↓
[신고 수수료 ₩40,000 결제]
   ↓
[사업장 소재지 시·군·구청 자동 이관]
   ↓
[1~2주 후 통신판매업 신고증 발급 → admin-web 법인 정보에 입력 → 3 콘솔 footer 자동 동기]
```

## 4. 구매안전서비스 이용확인증 (※ 핵심)

3 옵션 중 선택:

### A. PG 가맹 완료 시 (권장 — 가장 깔끔)
- **토스페이먼츠** 가맹점 신청 → 심사 완료 → 토스 관리자에서 "구매안전서비스 이용확인증" PDF 다운로드 → 정부24 첨부
- 가맹 신청 절차: <https://www.tosspayments.com> → 가맹점 신청

### B. PG 가맹 진행 중일 때
- 정부24 신고서에 "PG 심사 진행 중" 사유 기재
- 가맹 완료 후 시군구청에 추가 제출 (시군구청 담당자에게 사전 문의 필수)

### C. 매출 1년 1,200만원 미만 예상
- 「전자상거래법 시행규칙」 제8조 — 소규모 통신판매업자 면제 조건
- 면제 사유서 제출 (정부24 양식 다운 가능)
- 출시 초기 6개월 ~ 1년 매출 미확정 단계라면 가능

## 6. 사용자 입력 필요 항목 (⚠️ 표시 종합)

신고서 작성 전 다음 정보를 손에 두기:

1. ⚠️ **사업자등록번호 10자리**
2. ⚠️ **법인등록번호 13자리** (등기부등본)
3. ⚠️ **대표자 성명 + 주민등록번호**
4. ⚠️ **사업장 소재지 (전체 주소, 우편번호)**
5. ⚠️ **사업장 연락처**
6. ⚠️ **인터넷 도메인 TLD 결정** (pathwave.??? — 본 문서 작성 시점 미정)
7. ⚠️ **호스트 서버 소재지** (Contabo 싱가포르 / 한국 리전 추가 도입 여부)
8. ⚠️ **구매안전서비스 옵션 (A/B/C)** 결정 — 토스 가맹 진행 상태에 따라

## 7. 신고증 발급 후 PathWave 작업

| # | 작업 | 위치 |
|---|---|---|
| 1 | 통신판매업 신고번호 입력 | admin-web → 시스템 → 법인 정보 |

| 4 | Apple App Store 심사 자료 첨부 | App Store Connect 메타데이터 |
| 5 | Google Play 심사 자료 첨부 | Play Console 앱 정보 |

## 8. 위치기반서비스 사업 신고 (별도, 동시 진행 권장)

위치정보법 §9 — 본 PathWave 가 BLE 비콘 + GPS 사용하므로 **별도 신고 필수**.

| 항목 | 내용 |
|---|---|
| 신청 경로 | [방송통신위원회 전자민원](https://emnet.kcc.go.kr) 또는 정부24 |
| 비용 | 무료 |
| 처리 기간 | 2~4주 |
| 신고 분류 | "위치기반서비스 사업 신고" (개인위치정보 미수집 사업자) |

별도 가이드 문서 필요하면 요청 주시면 본 문서와 같은 형식으로 작성하겠습니다.
