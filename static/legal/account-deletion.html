PathWave 계정 삭제 안내

이 페이지는 Apple App Store 정책 5.1.1(v) 및 Google Play 정책에 따라 앱을 설치하지 않고도 계정을 삭제하고 데이터 처리를 요청하실 수 있도록 마련되었습니다.

1. 앱 안에서 직접 삭제 (권장)

  1. PathWave 앱 실행 → 우측 하단 설정
  2. 계정 관리 → 계정 삭제 선택
  3. 현재 비밀번호 재입력 → 확인
  4. 즉시 로그아웃되며 계정이 비활성화됩니다.

소셜 가입(카카오/구글/애플)은 비밀번호 입력 단계가 생략됩니다.

2. 이메일로 요청 (앱을 사용할 수 없는 경우)

아래 이메일로 가입 이메일·이름·요청 사유를 보내 주세요. 영업일 기준 3일 이내에 처리하고 결과를 회신해 드립니다.

privacy@pathwave.app 으로 메일 보내기

본인 확인을 위해 가입 시 사용한 이메일에서 보내 주시면 절차가 빨라집니다.

3. 삭제되는 데이터

  • 이메일·이름·전화번호·프로필 사진
  • 위치 사용 이력 / 비콘 감지 이력
  • 푸시 알림 토큰 (즉시 알림 차단)
  • 스탬프·쿠폰·즐겨찾기 등 본인 활동 데이터
  • 1:1 매장 채팅 메시지 — 본인 측 발신은 30일 후 익명화 / 매장 측 보존 (감사 목적)

4. 보존되는 데이터 (법령 의무)

  • 결제·환불 내역: 전자상거래법 5년
  • 로그인 기록: 통신비밀보호법 3개월
  • 부정이용 신고 처리 기록: 정보통신망법 6개월

5. 처리 흐름 (요약)

  1. 요청 접수 → users.deleted_at 즉시 기록 (소프트 삭제)
  2. 이메일 익명화 — 같은 주소로 재가입 가능
  3. 푸시 토큰 즉시 삭제 → 알림 중단
  4. 30일 grace 기간 (재로그인 시 복구 가능)
  5. 30일 경과 → 보존 의무 데이터 외 영구 삭제

6. 문의
