PathWave 청소년 보호 정책

본 정책은 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 제42조의2 및 「청소년 보호법」 에 따라 PathWave 가 만 14세 미만 아동을 포함한 미성년 사용자를 보호하기 위해 운영하는 방침입니다.

1. 만 14세 미만 가입 제한

회원 가입 시 출생년도를 입력받아 만 14세 미만은 가입을 거부합니다.

  • 가입 단계에서 자동 거부 → "보호자 동의가 필요합니다" 안내
  • 보호자가 본인 이메일로 동의해야만 가입 가능 (별도 보호자 초대 흐름)
  • 법령 개정 시 본 기준은 변경될 수 있습니다.

2. 보호자 동의 절차

  1. 만 14세 미만 사용자가 가입 시도 → 보호자 이메일 입력
  2. 보호자에게 동의 요청 메일 발송 (24시간 유효)
  3. 보호자가 링크 클릭 + 동의 → 자녀 가입 완료
  4. 보호자 거절 시 가입 차단

3. 청소년 유해 정보 차단

  • 주류 / 성인 전용 매장은 별도 분류 (adult_only=1)
  • 만 14세 미만 사용자에게는 성인 매장 목록 자동 제외
  • 채팅 메시지에 유해 표현 자동 필터 (욕설/혐오) + 신고 1건 즉시 검토 큐 진입

4. 개인정보 추가 보호

  • 만 14세 미만 사용자 정보는 보호자 동의 범위 내로 한정
  • 마케팅 정보 수신 동의는 만 14세 미만 → 자동 비활성
  • 위치 정보는 BLE 비콘 감지에만 사용, 외부 공유 X

6. 신고 및 문의

청소년 유해 정보 / 부적절한 사용자 또는 매장 발견 시:

  • 앱 안: 채팅 / 매장 상세 우상단 ⋮ → "신고" → 사유 선택 후 제출
  • 이메일: youth@pathwave.app

7. 외부 신고 채널

  • 경찰청 사이버수사국 (☎ 182)
  • 한국청소년상담복지개발원 (☎ 1388)
