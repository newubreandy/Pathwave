PathWave 쿠키 정책

본 정책은 PathWave 웹사이트(슈퍼어드민 / 사장 콘솔) 가 사용하는 쿠키·로컬 스토리지 등 추적 기술에 대한 안내입니다. 모바일 앱(iOS/Android) 은 OS 표준 스토리지 만 사용하며, 본 쿠키 정책 대상이 아닙니다.

1. 적용 범위

  • 슈퍼어드민 콘솔 (admin-web)
  • 매장 사장 콘솔 (provider-web)
  • 본 페이지가 포함된 정책 문서 (static)

2. 사용하는 추적 기술

종류이름용도보존 기간
HttpOnly 쿠키 session_id 로그인 세션 유지 (필수) 세션 종료 시
LocalStorage access_token / refresh_token JWT 인증 (필수) 로그아웃 또는 만료 시
LocalStorage i18nextLng 언어 설정 기억 (필수) 변경 시까지
LocalStorage theme / ui-prefs UI 환경설정 (선택) 변경 시까지

3. 광고 / 분석 트래킹

PathWave 는 광고 / 외부 분석 트래커 (Google Analytics / Facebook Pixel 등) 를 사용하지 않습니다.

iOS 앱 Privacy Manifest 의 NSPrivacyTracking=false 도 동일 정책을 반영합니다.

4. 쿠키 거부 시

  • 필수 쿠키 (session, JWT) 거부 → 로그인 불가
  • 선택 쿠키 (UI 환경) 거부 → 환경 설정이 기본값으로 매번 초기화

5. 제3자 쿠키

외부 결제(토스페이먼츠) 흐름 진입 시 토스 도메인에서 발급하는 쿠키는 토스페이먼츠 정책을 따릅니다.

6. 변경 통지

본 정책이 변경되면 admin/provider 콘솔 진입 시 알림이 표시되며, 7일 전 사전 공지합니다.
