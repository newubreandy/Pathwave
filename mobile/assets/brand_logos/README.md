# Brand Logos

여기에 각 SNS 의 **공식 브랜드 가이드라인** 에서 다운로드한 SVG 를 두면
자동으로 로그인 화면(SocialLoginRow)에 사용됩니다.

| 파일명 | 공식 다운로드 페이지 |
|---|---|
| `google.svg`   | https://developers.google.com/identity/branding-guidelines |
| `apple.svg`    | https://developer.apple.com/design/human-interface-guidelines/sign-in-with-apple |
| `facebook.svg` | https://about.meta.com/brand/resources/facebookapp/logo/ |
| `kakao.svg`    | https://developers.kakao.com/docs/latest/ko/reference/design-guide |
| `naver.svg`    | https://developers.naver.com/docs/login/bi/bi.md |

각 회사의 brand mark 는 **trademark** 보호 자산입니다.
반드시 위 공식 페이지에서 라이센스 조건을 확인한 뒤 다운로드해 사용하세요.

파일이 없으면 코드가 자동으로 `simple_icons` 패키지의 단색 fallback 으로 대체합니다.
